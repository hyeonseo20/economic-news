# 화면 설계안

> 작성일: 2026-04-29  
> 기준: 스마트폰 우선 (모바일 퍼스트), 로그인 불필요, 1~2분 이내 읽기

---

## 공통 화면 구성

모든 설계안은 아래 2개 화면을 공유한다.

- **메인 화면** — 오늘의 요약 (핵심 3줄 + 항목별 상세)
- **날짜 선택 화면** — 이전 요약 목록

---

## 설계안 A: 카드형 뉴스피드

> 콘셉트: 인스타그램 피드처럼 뉴스 항목을 카드로 쌓아 올린 구성.  
> 스크롤 한 번에 전체를 훑기 좋다.

### 메인 화면

```
┌─────────────────────────┐
│  📰 한경 오늘의 요약     │  ← 앱 타이틀
│  2026년 4월 29일 (화)   │
│             [날짜 선택 ▼]│
├─────────────────────────┤
│  ✦ 오늘의 핵심 3줄      │
│ ┌───────────────────┐   │
│ │ 1. 미 연준 금리 동결  │  │
│ │ 2. 삼성전자 실적 발표 │  │
│ │ 3. 원달러 환율 1,380  │  │
│ └───────────────────┘   │
├─────────────────────────┤
│  📌 뉴스 항목별 요약    │
│                         │
│ ┌───────────────────┐   │
│ │ 01 미 연준 금리 동결│   │  ← 카드
│ │ 연준은 이번 회의에서│   │
│ │ 금리를 동결하기로...│   │
│ │            [더보기]│   │
│ └───────────────────┘   │
│                         │
│ ┌───────────────────┐   │
│ │ 02 삼성전자 실적   │   │  ← 카드
│ │ 삼성전자가 1분기   │   │
│ │ 영업이익...       │   │
│ │            [더보기]│   │
│ └───────────────────┘   │
│           ...           │
│                         │
│ ┌───────────────────┐   │
│ │  🔗 링크 공유하기  │   │  ← 공유 버튼
│ └───────────────────┘   │
└─────────────────────────┘
```

### 날짜 선택 화면 (드롭다운)

```
┌─────────────────────────┐
│  날짜 선택              │
├─────────────────────────┤
│  ✔ 2026-04-29 (오늘)   │
│    2026-04-28           │
│    2026-04-25           │
│    2026-04-24           │
│           ...           │
└─────────────────────────┘
```

### 특징
- 장점: 스크롤 흐름이 자연스럽고 익숙한 피드 패턴
- 단점: 항목이 많으면 스크롤이 길어짐

## 설계안 B: 아코디언 리스트형

> 콘셉트: 뉴스 제목 목록이 먼저 보이고, 탭하면 상세 내용이 펼쳐지는 구성.  
> 원하는 뉴스만 골라 읽기 좋다.

### 메인 화면

```
┌─────────────────────────┐
│  한경 오늘의 요약        │
│  2026년 4월 29일 (화)   │
│             [날짜 선택 ▼]│
├─────────────────────────┤
│  오늘의 핵심             │
│  ① 미 연준 금리 동결    │
│  ② 삼성전자 실적 발표   │
│  ③ 원달러 환율 1,380    │
├─────────────────────────┤
│  전체 뉴스 (8건)        │
├─────────────────────────┤
│ ▶ 01  미 연준 금리 동결  │  ← 접힌 상태 (탭하면 펼침)
├─────────────────────────┤
│ ▼ 02  삼성전자 실적 발표 │  ← 펼친 상태
│                         │
│   삼성전자가 1분기 영업  │
│   이익으로 6.6조 원을    │
│   기록했다고 발표했다.   │
│   이는 전분기 대비...    │
│                         │
├─────────────────────────┤
│ ▶ 03  원달러 환율 동향   │  ← 접힌 상태
├─────────────────────────┤
│           ...           │
├─────────────────────────┤
│ ┌───────────────────┐   │
│ │  🔗 링크 공유하기  │   │
│ └───────────────────┘   │
└─────────────────────────┘
```

### 특징
- 장점: 제목 전체를 한눈에 파악 후 원하는 항목만 읽을 수 있음 (스터디 준비에 효율적)
- 단점: 탭 동작이 필요해 조작이 한 단계 더 있음

## 설계안 C: 신문형 레이아웃

> 콘셉트: 신문처럼 헤드라인이 크게 강조되고 본문이 아래로 이어지는 구성.  
> 경제 뉴스 특성에 맞는 전통적인 느낌.

### 메인 화면

```
┌─────────────────────────┐
│     한경 오늘의 요약     │
│   2026년 4월 29일 화요일 │
│             [이전 날짜 →]│
├─────────────────────────┤
│  [ TODAY'S BRIEF ]      │
│                         │
│  미 연준 금리 동결,      │  ← 헤드라인 (크고 굵게)
│  시장 안도 반응          │
│                         │
│  삼성전자 1Q 영업이익    │
│  6.6조 기록              │
│                         │
│  원달러 환율 1,380원대   │
│  유지                    │
├─────────────────────────┤
│  ─────── 상세 뉴스 ──── │
│                         │
│  ■ 미 연준 금리 동결     │  ← 섹션 제목
│                         │
│  연방준비제도는 28일(현지 │
│  시간) FOMC 회의를 열고  │
│  기준금리를 현행 수준인  │
│  5.25~5.50%로 동결하기로 │
│  결정했다. 파월 의장은   │
│  ...                    │
│                         │
│  ■ 삼성전자 실적 발표    │
│                         │
│  삼성전자는 29일 올해    │
│  1분기 영업이익이 6.6조  │
│  ...                    │
│                         │
├─────────────────────────┤
│ ┌───────────────────┐   │
│ │  🔗 링크 공유하기  │   │
│ └───────────────────┘   │
└─────────────────────────┘
```

### 특징
- 장점: 경제 뉴스 앱다운 분위기, 헤드라인이 강조되어 중요도 파악 쉬움
- 단점: 스크롤이 길고, 모든 내용이 펼쳐져 있어 원하는 항목 찾기 번거로울 수 있음

## 설계안 비교

| 항목 | A. 카드형 | B. 아코디언 | C. 신문형 |
|------|-----------|-------------|-----------|
| 1~2분 내 읽기 | ★★★ | ★★★★ | ★★★ |
| 원하는 항목만 읽기 | ★★★ | ★★★★★ | ★★ |
| 스터디 공유 후 가독성 | ★★★★ | ★★★ | ★★★★★ |
| 스마트폰 조작 편의성 | ★★★★ | ★★★ | ★★★★ |
| 경제 뉴스 분위기 | ★★★ | ★★★ | ★★★★★ |

---

## 권장안

**B (아코디언) + C (신문형) 혼합**

- 상단: 신문형처럼 헤드라인 3줄을 크고 굵게 표시
- 하단: 아코디언 방식으로 항목 탭 시 상세 내용 펼침
- 스터디 준비처럼 "제목 훑기 → 필요한 것만 읽기" 패턴에 최적화
