---

## US-01. 새 영상 자동 감지

**As a** 매일 아침 바쁜 직장인 준서로서,  
**I want** 한경 코리아마켓 플레이리스트에 새 영상이 올라오면 자동으로 감지되기를,  
**So that** 내가 직접 유튜브를 열어 확인하지 않아도 된다.

### 수락 조건
- [ ] 매일 오전 9시에 플레이리스트를 자동으로 확인한다
- [ ] 당일 업로드된 영상만 처리한다 (이전 영상 중복 처리 없음)
- [ ] 새 영상이 없는 날에는 아무 동작도 하지 않는다

---

## US-02. AI 요약 자동 생성

**As a** 영상을 직접 정리하는 데 1시간 이상 쓰고 있는 준서로서,  
**I want** AI가 영상 내용을 자동으로 요약해 주기를,  
**So that** 정리에 드는 시간을 없애고 스터디 공유에 집중할 수 있다.

### 수락 조건
- [ ] 영상 자막을 기반으로 요약을 생성한다
- [ ] 상단에 **오늘의 핵심 3줄 요약**이 표시된다
- [ ] 하단에 **뉴스 항목별 제목 + 상세 설명 (2~3문단)**이 표시된다
- [ ] 요약은 한국어로 작성된다
- [ ] 요약 생성은 오전 9시 이전에 완료된다

---

## US-03. 스마트폰 웹에서 요약 확인

**As a** 스마트폰으로 뉴스를 보는 준서로서,  
**I want** 링크 하나만 열면 오늘의 요약을 바로 볼 수 있기를,  
**So that** 업무 시작 전 1~2분 안에 핵심 내용을 파악할 수 있다.

### 수락 조건
- [ ] 고정된 URL 하나로 항상 최신 요약에 접근할 수 있다
- [ ] 날짜별로 이전 요약을 목록에서 선택해 볼 수 있다
- [ ] 스마트폰 브라우저에서 읽기 편한 레이아웃이다
- [ ] 페이지 로딩 속도가 3초 이내이다

---

## US-04. 요약 완료 시 푸시 알림 수신

**As a** 업무 시작 전에 뉴스를 챙기고 싶은 준서로서,  
**I want** 요약이 준비되면 스마트폰에 알림이 오기를,  
**So that** 내가 직접 페이지를 열어보지 않아도 새 요약이 생겼음을 알 수 있다.

### 수락 조건
- [ ] 요약 생성 완료 시 즉시 브라우저 푸시 알림이 전송된다
- [ ] 알림 메시지에 오늘 날짜와 "요약이 준비됐습니다" 문구가 포함된다
- [ ] 알림을 탭하면 요약 웹 페이지로 바로 이동한다
- [ ] 새 영상이 없는 날에는 알림이 오지 않는다

## US-05. 뉴스 책갈피 저장

**As a** 경제스터디를 준비하는 준서로서,  
**I want** 중요한 뉴스 항목을 책갈피로 저장해 두기를,  
**So that** 나중에 다시 찾아보거나 스터디에서 꺼내 쓸 수 있다.

### 수락 조건
- [ ] 각 뉴스 항목 제목 우측에 책갈피 버튼이 표시된다
- [ ] 책갈피 버튼을 클릭하면 저장되고 아이콘이 활성화된다
- [ ] 다시 클릭하면 저장이 해제된다
- [ ] 저장된 뉴스는 MY PAGE 탭에서 목록으로 확인할 수 있다
- [ ] 페이지를 새로고침해도 저장 목록이 유지된다

---

## US-06. MY PAGE에서 저장 목록 확인

**As a** 여러 날의 뉴스를 관리하고 싶은 준서로서,  
**I want** MY PAGE 탭에서 저장한 뉴스들을 한눈에 보기를,  
**So that** 스터디 전에 저장해 둔 내용을 빠르게 복습할 수 있다.

### 수락 조건
- [ ] MY PAGE 탭에서 저장된 뉴스 목록이 표시된다
- [ ] 각 항목에 날짜와 뉴스 제목이 표시된다
- [ ] 저장된 항목이 없을 때는 안내 메시지가 표시된다
- [ ] 항목을 탭하면 상세 내용을 확인할 수 있다

## US-07. 무료로 지속 운영 (구 US-06)

**As a** 개인 비용을 최소화하고 싶은 준서로서,  
**I want** 앱이 별도 비용 없이 운영되기를,  
**So that** 부담 없이 매일 사용할 수 있다.

### 수락 조건
- [ ] 모든 구성 요소가 무료 티어 내에서 운영된다
- [ ] 월 운영 비용이 $0이다
- [ ] 무료 한도 초과 위험이 생기면 사전에 인지할 수 있다
