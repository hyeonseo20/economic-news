# Economic News App 요구사항 문서

> 작성일: 2026-04-29 | 상태: 확정

## 개요
한경 코리아마켓의 "한국경제신문 30분 만에 읽기" 플레이리스트에 매일 올라오는 영상을
AI가 자동으로 요약해 웹 페이지에 보여주는 앱.

---

## 확정된 요구사항

### 콘텐츠 소스
- **채널:** 한경 코리아마켓
- **플레이리스트:** 한국경제신문 30분 만에 읽기
- **URL:** https://www.youtube.com/playlist?list=PLVups02-DZEWWyOMyk4jjGaWJ_0o1N1iO

### 요약 형식
- 상단: **오늘의 핵심 3줄 요약**
- 하단: **뉴스 항목별 상세 설명** (제목 + 2~3문단)

### 확인 방법
- **웹 브라우저**에서 페이지를 열어 확인
- 매일 **오전 9시**에 요약 완료
- ~~링크 공유 기능~~ — 제거됨

### 탭 메뉴
- 헤더 하단에 **NEWS** / **MY PAGE** 탭 2개
- **NEWS 탭:** 썸네일 + 핵심 요약 + 뉴스 항목별 아코디언 + 항목별 책갈피 버튼
- **MY PAGE 탭:** 책갈피로 저장한 뉴스 목록 표시

### 알림
- **브라우저 푸시 알림** — 요약 완료 시 폰/PC에 팝업 전송

### 비용
- **무료 운영** 목표 (유료 서비스 최소화)

---

## 기술 설계 방향 (초안)

| 역할 | 도구 | 비용 |
|------|------|------|
| 영상 감지 | YouTube Data API v3 | 무료 (일 10,000 쿼터) |
| 자막 추출 | youtube-transcript-api | 무료 |
| AI 요약 | Google Gemini API (Flash) | 무료 티어 존재 |
| 웹 페이지 호스팅 | GitHub Pages | 무료 |
| 자동 실행 스케줄러 | GitHub Actions | 무료 (월 2,000분) |
| 푸시 알림 | ntfy.sh | 무료 |

### 동작 흐름
```
매일 오전 9시
→ GitHub Actions 자동 실행
→ YouTube API로 신규 영상 확인
→ 자막 추출
→ Gemini AI로 요약 생성
→ 요약 HTML 파일 생성 → GitHub Pages 배포
→ ntfy.sh로 푸시 알림 전송
→ 사용자가 브라우저에서 확인
```
